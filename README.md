# BeautifulSoup web scraper.
BeautifulSoup4 is a python library for scraping information from web pages. This repo contains one program designed to scrape any text a user may want, and two example programs to demonstrate the power of a good soup for scraping text.
## bs4-scraper
This program will ask a user for a target URL to scrape text from. After the user has provided the URL they would like to scrape, the program will prompt the user for the tag and class of the desired text. The program will then return the user's requested text as standard output.
## quote-scraper
This program provides the user with a list of quotes, scraped from quotes.toscrape.com
## news-scraper
This program will provide the user with a list of recent headlines from cybersecurity blogs and news sites.
